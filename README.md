# bonjourplatane.fr

Tous les jours, une image de platane.

## Requirements

- ruby 3.0.2

## Install

```sh
bundle install
```

Check `.env.local` for env

## Run in local

```sh
./bin/local.sh
```

Then visit `http://127.0.0.1:4000/`

## Generate today post

Run:

```sh
bundle exec ruby bin/generate_platane_post.rb
```
